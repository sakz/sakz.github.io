亚马逊AWS更改LINUX为ROOT权限密码登陆

  • aws
  • linux

我的AWS VPS的LINUX版本是UBUNTU 14.04，首先用AWS证书验证的账户登录.

1、修改ROOT密码

sudo passwd root

2、修改sshd_config文件

sudo vim /etc/ssh/sshd_config

PermitRootLogin
这行改为
PermitRootLogin yes

PasswordAuthentication no
上面的no改为yes
UsePAM yes
上面的yes改为no
修改完保存退出。

3、重启AWS VPS，就可以使用root正常登陆了。
