finalspeed客户端简易使用教程

  • proxy

0.首先SS账号的服务器要安装了finalspeed,并开启了。

安装脚本

1.windows平台

1.FinalSpeed客户端Windows版

下载地址:finalspeed_install1.2.exe

2.Mac平台

FinalSpeed客户端Java版,支持OS X,Linux.

下载地址:finalspeed_client1.2.zip

系统需安装java运行环境,Linux还需安装libpcap.

Ubuntu,Debian安装libpcap: apt-get -y install libpcap-dev

Centos安装libpcap: yum -y install libpcap

安装:下载解压.

运行方法:
方法1：直接双击finalspeed_client.jar

方法2：打开终端,假设finalspeed_client.jar所在路径为/fsclient ,先切换到该路径cd /fsclient ,然后执行 sudo java -jar finalspeed_client.jar ,前面加sudo,因为必须以root权限运行,如果没有root权限,会无法启用tcp协议

3.注意问题,必读!

1.服务器必须同时部署FinalSpeed服务端才能进行加速.

2.客户端必须准确设置物理带宽,最终加速的速度不会超过所设置的带宽值,如果设置值高于实际带宽会造成丢包,导致速度变慢.

3.客户端使用UDP协议.

4.使用方法

假设服务器IP为us3.fazero.pw,ss端口为25000.加速前提ss服务端运行正常,ss客户端也能正常登录.
finalspeed客户端配置和ss客户端配置如下图,配置好后打开google.com，状态才会有显示。

注意ss客户端的服务器IP必须为127.0.0.1 finalspeed客户端的地址填你的ss服务器IP.
